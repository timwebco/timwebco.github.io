## Getting Started

Simply open `index.html` in your web browser to view the portfolio.

### Customization

1. Update personal information in `index.html`:
   - Replace "Your Name" with your actual name
   - Update project descriptions and tags
   - Add your contact information (email, GitHub, LinkedIn)

2. Customize colors in `styles.css`:
   - Edit CSS variables in `:root` selector
   - Modify the gradient colors in hero and project sections

3. Add/remove sections as needed
